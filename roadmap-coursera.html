SEM1
Python Crash Course — Base para todo lo demás
Google — Crash Course on Python · Sin experiencia previa
~15 hrs

No puedes construir agentes sin Python. Pero la buena noticia: no necesitas ser experto — solo necesitas lo suficiente para trabajar con LangChain. Este curso de Google te lleva de cero a escribir scripts funcionales en 2 semanas.

🐍 Crash Course on Python — Google Coursera 📚 Programming for Everybody — U. Michigan (alternativa más suave) Coursera

Python for Everybody — APIs y Web Data · Michigan
~8 hrs

Para construir agentes necesitas entender cómo funcionan las APIs REST y el formato JSON — que es la "lengua" en que hablan los modelos de IA. Esta semana dominas eso.

🌐 Using Python to Access Web Data — U. Michigan Coursera
💡 Al terminar esta semana: haz tu primera llamada a la API de OpenAI o Anthropic desde Python. Esto es literalmente la base de todos los agentes.

SEM4
LangChain: el framework para construir agentes
DeepLearning.AI — LangChain for LLM App Dev · Harrison Chase
~6 hrs

LangChain es el framework más popular para conectar LLMs con herramientas externas. Este curso lo da el mismísimo creador de LangChain junto a Andrew Ng. Chains, memoria, agentes simples.

⛓️ LangChain for LLM Application Development — DeepLearning.AI + Harrison Chase Coursera
💡 Proyecto de semana: Construye un agente que lea PDFs de reportes de tu empresa y responda preguntas sobre ellos. Esto es RAG básico y es inmediatamente útil.

Fase 2 — Construir agentes reales · Semanas 5–8
SEM5
Fundamentos de AI Agents con LangChain
IBM — Fundamentals of Building AI Agents · Coursera
~8 hrs

Diseña agentes que razonan, toman acciones y usan herramientas externas. Tool calling, LCEL, agentes que ejecutan SQL, generan visualizaciones y analizan datos con lenguaje natural.

🤖 Fundamentals of Building AI Agents — IBM Coursera
💡 Proyecto: Agente que conecte a tu base de datos y responda preguntas como "¿cuáles fueron los 5 DAGs más lentos esta semana en Airflow?" en lenguaje natural.
SEM6
RAG: Agentes con memoria y documentos propios
IBM — Fundamentals AI Agents using RAG and LangChain
~8 hrs

Retrieval-Augmented Generation (RAG) te permite conectar tu agente con documentación interna, manuales técnicos, políticas de empresa o historial de errores de Airflow. El agente aprende de tus propios datos.

🗄️ Fundamentals of AI Agents using RAG and LangChain — IBM Coursera
💡 Proyecto clave: Agente que indexa toda la documentación técnica de tu equipo y responde preguntas de onboarding para nuevos desarrolladores automáticamente.
SEM7
LangGraph: agentes con flujos y estados
DeepLearning.AI — AI Agents in LangGraph · Harrison Chase
~8 hrs

LangGraph es la evolución de LangChain para agentes complejos: flujos condicionales, memoria persistente entre conversaciones, human-in-the-loop y workflows similares a Airflow pero con IA. Fundamental para automatizar revisión de código.

🕸️ AI Agents in LangGraph — DeepLearning.AI + Harrison Chase Coursera
💡 Proyecto estrella: Agente de revisión de código con LangGraph: recibe un PR, analiza el diff, sugiere mejoras, verifica estándares del equipo y genera un reporte automático.
SEM8
Agentic AI con LangChain y LangGraph — IBM
Sistemas multi-agente · ReAct · Reflexion · Memoria
~10 hrs

Arquitecturas avanzadas de agentes: Reflection (el agente evalúa y mejora su propio output), ReAct (razona antes de actuar) y multi-agente (varios agentes especializados colaborando). Ideal para automatización compleja.

⚡ Agentic AI with LangChain and LangGraph — IBM Coursera
💡 Aquí ya puedes construir: un agente que monitorea tus DAGs de Airflow, detecta fallos, diagnostica la causa y propone solución — todo automáticamente.

🔍 Referencia: Agentic AI con LangGraph — IBM (casos de tool calling) Coursera
